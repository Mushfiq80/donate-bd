Important FAQs

1. What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to access and manipulate HTML and XML documents programmatically using languages like JavaScript.

2. How do you select an element from the DOM?

You can select an element from the DOM using various JavaScript methods, such as document.getElementById() to select by ID, document.querySelector() to select using CSS selectors, or document.getElementsByClassName() to select elements by class name.

3. How can you change the content of an element in the DOM?

You can change the content of an element by accessing it and modifying its innerHTML or textContent property. For example: document.getElementById('example').textContent = 'New Content';.

4. What is event handling in the DOM?

Event handling in the DOM refers to the process of detecting and responding to user actions (events) such as clicks, mouse movements, or keyboard inputs. You can add event listeners to elements using methods like addEventListener(). For example: document.getElementById('btn').addEventListener('click', function() { alert('Button clicked!'); });.
